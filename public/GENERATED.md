# Generated Public Registry Files

These files are generated by `npm run publish:registry` from reviewed records in `data/public-profiles.json`:

- `public/registry.json`
- `public/agents/index.json`
- `public/agents/*.json`
- `public/sitemap.xml`
- `public/robots.txt`

Keep these paths stable because agents and crawlers use them directly. Do not manually add private dashboard data, API secrets, private negotiation rules, hidden minimum prices, or internal notes to generated files.

To publish reviewed records:

```bash
npm run publish:registry
```
